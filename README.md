# LAUREL : : CERTIFICATE GENERATOR BASED ON TEMPLATES

- FORK THIS PROJECT
- CLONE IT
- RUN THE FOLLOWING
```
cd laurel
npm install
```
- CREATE A FIREBASE PROJECT AND SET UP HOSTING
- CHANGE THE `firebaseConfig` in `src/firebase.js`
- RUN THE FOLLOWING
```
npm start
```

THIS WILL GET THE PROJECT RUNNING.
